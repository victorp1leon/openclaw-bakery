# Quickstart

Status: Draft
Last Updated: 2026-03-03

## Local Setup
1. Instalar dependencias:
   - `npm install`
2. Crear/ajustar `.env` segun `documentation/operations/config-matrix.md`.
3. Ejecutar runtime local:
   - `npm run dev`
4. Validar tests:
   - `npm test`
5. Ejecutar validacion minima de seguridad:
   - `npm run verify:security`
