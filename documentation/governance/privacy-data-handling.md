# Privacy and Data Handling

Status: Draft
Last Updated: 2026-03-03

## Baseline Rules
1. Minimizar recoleccion de datos sensibles.
2. No loggear secretos ni payloads sensibles sin redaccion.
3. Definir retencion de datos operativos y respaldo seguro.
4. Restringir acceso por necesidad operacional.
